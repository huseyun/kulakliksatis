# Kulaklik Satis API

## Proje Tanimi
Bu proje, kulaklık satışı yapan bir e-ticaret platformunun arka yüz servislerini sağlamak amacıyla geliştirilmiştir. Sistem; yönetici, satıcı ve müşteri olmak üzere üç farklı kullanıcı tipi için özelleştirilmiş işlevler sunar. Kullanıcı yönetimi, ürün listeleme ve güvenli kimlik doğrulama süreçlerini kapsayan RESTful bir API yapısına sahiptir.

## Proje Kategorisi
E-Ticaret ve Arka Yüz (Backend) Servisleri

## Referans Uygulamalar
Bu çalışma hazırlanırken standart e-ticaret sistemleri, Spring Security mimarisi ve modern REST API tasarım prensipleri referans alınmıştır.
